bool Log_viewer::NRG_legacy_log_format::match ( const QString  first_line) const
inlinevirtual

This method returns true if we can match the passed line.

A regular expression is used to match the first_line. NRG lecacy format expects the line to start with a date formated like this:

2010-09-08 12:06:12,079

and shall be followed by a SPACE and a pipe char after the date.
